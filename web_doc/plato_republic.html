《理想国》西方政治哲学与秩序博弈体系

Πολιτεία · Plato · 约公元前380年 · 西方哲学奠基之作
正义·灵魂·城邦·哲人王 · 纯Canvas可视化 · 零外部依赖

《理想国》（The Republic）的终极追问——柏拉图（苏格拉底）在全书开篇即抛出一个令所有对话者从轻松寒暄坠入严峻思辨的终极问题：「什么是正义？」是好人的正义者在任何情况下都比不正义者更幸福吗？《理想国》全书十卷围绕这两个巨大的核心问题逐层展开：从个人的正义探讨到城邦的正义——再从城邦的正义返照个人的灵魂结构——最终在「哲人王统治的理想城邦」和「灵魂三分秩序」的双线对标中给出回答：正义=灵魂或城邦中每个部分「各司其职、互不越位」的内在和谐秩序。

一、正义之问——全书一切论证的出发点

正义（Δικαιοσύνη / Dikaiosynē）——柏拉图借苏格拉底之口，在《理想国》第一卷中对「正义」首先进行了否定之路：逐个检验并拒绝当时希腊社会对正义的三种主流定义。他拒绝的不是定义本身的内容，而是每一类定义背后的逻辑漏洞。

1.1 三个被逐一驳倒的正义定义

苏格拉底在《理想国》卷一中逐一驳斥的三个正义观： ① 克法洛斯(Cephalus)的正义观——「正义=说真话和偿还所欠之物」 苏格拉底的辩证法攻击：「如果借给你武器的朋友突然发疯了——把武器还给他还算是 正义吗？将这个疯人的武器还给他意味着放任他伤害自己。」→规则不能无条件适用 → 正义≠机械地执行某套固定的规则 ② 波勒马库斯(Polemarchus)的正义观——「正义=把对每个人的应得给予他 帮助朋友、损害敌人」 苏格拉底：「如果对敌人采取伤害——不正是使他们变得更不正义吗？正义者能使任何人 变得更不正义吗？如果正义的功能是使所有人变得更正义——伤害与正义就互相排斥。」 →「损害敌人」本身是一种不正义的行为→正义者不可能以不正义为手段 ③ 特拉叙马库斯(Thrasymachus)的正义观——「正义=强者的利益」 「每一个政体下的法律都是按照统治者的利益制定的——因此无论在哪，正义都是 强者的利益。」这是全书中对正义最辛辣、最现实主义的挑战。 苏格拉底的反驳：任何技艺本身(医疗·航海·统治)的目的=为技艺所作用的对象服务 —不是为技艺者自身的利益服务。医生=为病人的健康、船长=为船客的安全、 真正的统治者=为国家公民的福祉→「统治者的天然义务=让被统治者获益」

📋 第一模块复盘

  • 正义的否定之路：否定了规则主义·报复主义·强权主义三种正义观
  • 特拉叙马库斯的挑战是全书最严峻的论敌——「正义=强者获益」
  • 苏格拉底的元论点：技艺的本性是为对象服务→不是为技艺者获利

二、灵魂三分理论——人的内心是个微型城邦

灵魂三分（Tripartite Soul）——柏拉图在卷四提出的核心心理学模型：人的灵魂内部不是浑然一体，而是由三个各自具有不同欲求和功能的部分组成的复合结构。三者之间持续的紧张与调和构成了每个人的内在精神景观。

2.1 灵魂的三个部分

柏拉图的灵魂三分结构： 理性部分（τὸ λογιστικόν / Logistikon · Reason） 功能：推理·计算·追求真理·在灵魂内部发号施令 对应的德性：智慧（Σοφία / Sophia） 在城邦中的投射：护卫者—统治者 激情部分（τὸ θυμοειδές / Thymoeides · Spirit） 功能：荣誉感·愤怒·竞争精神·挺身而出维护正义的冲动 对应的德性：勇敢（Ἀνδρεία / Andreia） 在城邦中的投射：辅助者—战士 关键特征：激情天然是理性的盟友——一个正义的灵魂中，激情听从理性的命令， 一同约束永不满足的欲望。如果把理性比作马车夫，激情是其中的白马。 欲望部分（τὸ ἐπιθυμητικόν / Epithymetikon · Appetite） 功能：饥饿·性欲·金钱欲·对物质满足的无尽渴望 对应的德性：节制（Σωφροσύνη / Sophrosyne）——即欲望安守其分、不试图僭越 在城邦中的投射：生产者—农民·工匠·商人 关键特征：欲望在本质上不是「邪恶」——它是灵魂正常运作的必需成分。恶化的发生 在于欲望不受理性和激情的制约，反客为主接管整个灵魂的控制权
图1：柏拉图灵魂三分结构模型——理性(逻各斯)·激情(θυμος)·欲望(επιθυμια)三者的等级秩序对照城邦三分，基于《理想国》卷四核心论述复刻。核心推演：正义的灵魂状态=理性通过激情的协助向欲望下达指令、欲望在理性的引领下安守其分。三者各安其位→灵魂和谐=正义的实现。不正义=欲望越过理性和激情篡夺了灵魂的最高统治权。

📋 第二模块复盘

  • 灵魂=理性(驭者)+激情(白马·荣誉)+欲望(黑马·物质)
  • 正义灵魂=理性通过激情约束欲望→三者各司其职
  • 不正义=欲望反客为主→灵魂的僭主状态

三、理想城邦的阶层构造——灵魂三分的外在投影

理想城邦（Καλλίπολις / Kallipolis）——柏拉图将人的灵魂三分结构「用大字写出来」，投射到整个城邦的宏观政治图景中。正如灵魂中的理性应当统治激情和欲望，城邦中也应当由一个理性发展到最高程度的小群体——哲人王——来施行统治。

3.1 城邦三阶层的功能与品性

理想城邦的三个功能阶层——与灵魂三分的精准对映： 第一阶层：护卫者—统治者（Φύλακες / Guardians·哲人王） 灵魂侧：纯理性 制造材料：金 品性：智慧——对什么是「真的」有最高洞察 教育：通过几十年的数学·辩证法·天文学训练→最终看到「善的理型」 义务：在轮值期治理城邦、以理性引导全部城邦事务 第二阶层：辅助者（Ἐπίκουροι / Auxiliaries·战士） 灵魂侧：激情 制造材料：银 品性：勇敢——不论危险或诱惑，永远坚守理性所告知的「什么是可以畏惧的」 义务：对内维持城邦秩序、对外守卫城邦免受外敌入侵 第三阶层：生产者（Δημιουργοί / Producers·农工商） 灵魂侧：欲望 制造材料：铁和铜 品性：节制——安于自身岗位、不试图逾越阶级向上爬 义务：从事农业·手工业·商业——生产城邦所需的全部物质供应 城邦正义=每个阶层「专注自己分内的事，不越权插手其他阶层的事务」 → 生产者不可进入统治地位(否则=欲望统治了理性→灵魂↔城邦双重不正义) → 护卫者不可参与生产经商(否则=理性被欲望污染→无法保持公正无偏) → 良好的城邦运作如同良好的灵魂运作——各部门各司其职→全体和谐
图2：理想城邦三阶层结构模型——统治(金·理性·智慧)·辅助(银·激情·勇德)·生产(铁铜·欲望·节制)，基于《理想国》卷三-四核心论述复刻。核心推演：城邦正义=三个阶层各安其位、不越级不交叉。每个阶层对应灵魂的一个部分、一种金属、一种核心德性。社会阶层的不平等在柏拉图框架中不是「压迫」，而是「天然的质料差异」——灵魂中理性强的人天生适合统治。

📋 第三模块复盘

  • 城邦=灵魂的大字版→统治者(理性)·辅助(激情)·生产(欲望)
  • 正义城邦=各阶层「做自己分内的事，不越位」
  • 金属神话(金银铜铁)=阶层遗传的教育奠基叙事

四、护卫者的教育——一个长达五十年的灵魂塑造工程

护卫者的教育体系（The Education of the Guardians）——柏拉图用了《理想国》卷二和卷三的大量篇幅描述如何培养未来的统治者。这不是「传授知识」，而是整个人的灵魂从幼年到成年的完整塑造——一个经过精心设计的、长达数十年的「制造理性最高统治者」的教育蓝图。

4.1 护卫者教育的两阶段

柏拉图教育体系的二阶段构造： 第一阶段：体育与音乐（0-20岁）——身体与灵魂的基础训练 «音乐»(μουσική / Mousikē)：在教育体系中包括文学·诗歌·音乐—— 护卫者从小只允许接触「赞美美德与神祇」的故事和曲调；任何描绘神祇 作恶、英雄在哭泣和恐惧的诗歌都被绝对禁止——因为儿童最早吸收的一切 烙印了其一生对善恶和对宇宙秩序的全部基本认知。 → 柏拉图对诗人(荷马·赫西俄德)的驱逐=不是反对诗歌的美——而是反对诗歌中的虚假道德观 第二阶段：数学·天文学·辩证法（20-50岁）——三十年逐梯爬向善的理型 20-30岁: 全面的数学训练——算术·几何·天文·和声——训练灵魂将眼睛从 感性世界扭向纯理性世界 30-35岁: 辩证法训练——从前提推论到最高原理、从最高原理向下还原全部知识 35-50岁: 投身公共事务——在战争的实务中检验理论→将知识锻入实际经验 50岁：经过上述全部阶梯训练的极少数的个体——「看到了善本身」——被要求返回洞穴轮流统治城邦
图3：理想国护卫者教育体系——从幼年到50岁的完整阶梯路径（音乐→体育→数学→天文→辩证法→实践→看见善→回洞穴统治），基于《理想国》卷七教育和洞穴寓言复刻。核心推演：教育≠填充知识——是「灵魂转向」(περιαγωγή / Periagoge)整个过程——将灵魂从感性的可见世界一层层转向理性的可知世界→最终望见一切存在的最高原理=善的理型→此后回到世间以「看见了终极真理」的眼光统领城邦。

📋 第四模块复盘

  • 护卫者教育=完整灵魂转向术→从感性→理性→善的理型
  • 驱逐诗人=防止虚假叙事污染幼年灵魂的道德底色
  • 五十岁=最终「看见善自身」→然后返回洞穴轮流治理

五、哲人王与洞穴寓言——全书的巅峰与转折

哲人王（Φιλόσοφος-Βασιλεύς / Philosopher-King）——柏拉图在卷五提出的所有政治哲学中最著名（也最具争议）的论断：「除非哲学家成为国王，或者国王成为真正的哲学家，否则城邦的苦难不会终结。」这不是一个权力建议——而是一个逻辑必然：除非治理城邦的人「知道什么是真的、什么是善的」，否则任何政体都注定以不同的形式重复同样根本的不义。

5.1 洞穴寓言——人类认知困境的终极隐喻

柏拉图洞穴寓言(卷七)——全部西方思想的奠基隐喻： 一群人从小被锁在一个洞穴中，只能看到洞壁上由身后火光投射的皮影戏 → 他们认为墙上的影子=全部现实——他们用一生互相比较谁能最快辨识影子的先后 某一天，其中一个囚徒被解开锁链→他转过身→被火光晃得睁不开眼→ 慢慢适应→他看到影子来源的木偶机关→意识到此前一生所认为的全部「现实」都是幻影 然后他被迫被拖出了洞穴→直接面对太阳→被阳光灼得什么也看不见→ 渐渐眼睛适应→他开始直接看到真实的事物——树木·云朵·天空→最后看到太阳本身 → 他知道太阳是万物可见与生长的终极原因——以前的洞穴中所见的一切都是极度贬值的映像 最难的时刻：他需要返回洞穴。习惯了阳光的眼睛在绝对黑暗中什么都看不清→ 洞中同伴嘲笑他出去一趟把眼睛都弄瞎了——他们拒绝被解救——谁要试图解开他们的锁链， 他们就会杀了他（参照苏格拉底之死）。 哲人王的含义就在于此：那少数走出了洞穴、亲眼看见了「善自身」的人， 必须返回洞中——不是因为他们想，而是因为正义要求他们去。在洞中他们要被嘲讽、 误解、最后被处死。但哲人王治理城邦不是因为「他一己的私利可以从中获得满足」， 而恰恰相反——是背负着沉重的正义义务——以『看见了太阳』的灵魂去统领黑暗中的城邦。
图4：柏拉图洞穴寓言认知层级模型——从影子的影像→可见实物→数学思考→理型→善自身，基于《理想国》卷六-卷七的线喻和洞穴寓言复刻。核心推演：人类认知有四个层级的最深层假设——最低层=感觉认定的影子世界(大众一生停留于此)；最高层=理性洞察的「善的理型」(善=全部实在的终极原因→哲人王之所见)。走出洞穴=灵魂的转向；返回洞穴=正义的实现。洞穴寓言是全部西方哲学中地位最高的元隐喻。

📋 第五模块复盘

  • 洞穴=人类根本生存困境的隐喻：一生把影子当真实
  • 哲人王=走出了洞穴、看见了太阳(善)、必须回洞治理
  • 苏格拉底之死=返回洞穴的哲人被洞中囚徒杀死的第一个实例

六、政体演化循环——五种政体逐级堕落的全过程

政体演化循环（Decline of Regimes）——柏拉图在卷八和卷九中描述了政体从最优(贵族/哲人王制)向最劣(僭主制)逐级堕落、不可逆的五阶段退化过程。每一个政体都是从前一个政体中由内部矛盾自然滋生长出的——没有人有意「选择」堕落，但上一个政体的根本缺陷必然孕育出下一个更劣的政体。

6.1 五政体的逐级堕落逻辑

五种政体按恶化顺序排列——每一级的缺陷都自动生出下一级： ① 贵族/哲人王制（Ἀριστοκρατία / Aristocracy）——最优 统治原则：理性·智慧。由最优秀的、看见善自身的少数人所统治。 「贵族」=「最优者之治」(不是血缘贵族——是德性和智慧的贵族) 对应的灵魂状态：理性统御激情和欲望→灵魂内部的和谐秩序 堕落原因：完美的数学周期被算错→护卫者阶级开始不再能精确识别优秀的下一代 ② 荣誉政体（Τιμοκρατία / Timocracy）——从理性到荣誉的坠落 统治原则：荣誉·声望·军功 斯巴达和克里特=希腊世界两例荣誉政体 对应的灵魂状态：激情(荣誉心)取代理性→欲望被压制但从未消失 ③ 寡头政体（Ὀλιγαρχία / Oligarchy）——从荣誉到钱财的坠落 统治原则：财富。政权掌握在最富有的人手中→财富资格线取代能力和德性 致命缺陷：富人和贫人的分裂被永久化+制度化→一个城邦不再是「一个城邦」 而是「富人的城邦」和「穷人的城邦」两个互相敌视的群体被迫捆在一起 ④ 民主政体（Δημοκρατία / Democracy）——从钱财到自由的坠落 统治原则：自由与平等。寡头制下有钱人和穷人对抗到极限→穷人揭竿而起→ 杀了一些富人、赶走了另一些→建立了人人平等的公民权 缺陷：将平等推至逻辑极限→自由从「有序的自由」变为「每个人都为所欲为」 →柏拉图：「驴子在街走没人让路，它昂首阔步觉得自己完全是自由的」 → 民主政体最脆弱在「缺乏秩序的自由」→产生无法节制的欲望放纵 ⑤ 僭主政体（Τυραννίς / Tyranny）——从自由到极权奴役 民主的失序→民众呼唤「能整饬秩序的强人」→一个煽动家(Protector)被拥戴上台 →他首先以保护民众为名清洗全部旧势力→又为防自己被害建立私人卫队→ 最后演变成绝对权力的僭主→以恐惧和暴力奴役整个城邦 → 僭主=最不正义的人。他的灵魂被最低级的欲望所禁锢。表面上的权力顶峰 实为精神最悲惨、最恐惧的奴役状态
图5：柏拉图政体演化五阶段循环——贵族→荣誉→寡头→民主→僭主，基于《理想国》卷八-九核心论述复刻。核心推演：每一种政体的根本缺陷是它前一个政体中尚未解决的内在张力→从最优到最劣的演化如同斜坡上的球——没有人故意推它，重力(灵魂内部欲望→激情→理性每一级的失守)是自我驱动的。柏拉图在公元前380年预言了雅典民主在伯罗奔尼撒战败后的自我残杀(苏格拉底之死是其血证)。

📋 第六模块复盘

  • 五政体堕落链：贵族→荣誉→寡头→民主→僭主→内部自我驱动
  • 民主的致命弱点：自由无度→欲望放纵→呼唤强人→极权奴役
  • 僭主=表面上最强大、精神上最悲惨的最不正义的人

七、个人正义与城邦正义——灵魂与城邦的同构对应

灵魂→城邦的同构映射（Isomorphism of Soul and Polis）——柏拉图在卷四提出《理想国》最核心的方法论：「正义可以是一个人的德性，也可以是一个城邦的德性。城邦比个人大——我们可以在城邦的层面上寻找正义，然后再把发现的正义写小——还回到个人灵魂中。」

7.1 灵魂-城邦对照表

柏拉图的双层正义对照结构： 个人灵魂 ←———→ 理想城邦 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 理性 ———→ 护卫者-统治者 (追求真理和智慧) (哲人王以理性统领一切城邦事务) 激情 ———→ 辅助者-战士 (荣誉感·捍卫正义) (勇敢地保卫城邦内外秩序) 欲望 ———→ 生产者 (物质需求·享乐) (安守其职、提供全部物质给养) 个人正义=每个灵魂部分各司其职 「理性以智慧统领，激情以勇敢支持理性，欲望以节制安守其份」 → 正义者的灵魂内部是和谐的——他不是靠外部力量被强制做正义的事， 而是他的灵魂内部各元素的秩序本身使得他不义是不可欲的 城邦正义=每个阶层各司其职 → 生产→辅助→统治三层不互相僭越→城邦和谐→正义实现 苏格拉底最后一击：正义者一定比不正义者更幸福 正义的人=灵魂和谐——他即使在贫困和受难之中仍有内在的秩序与平安 不正义的人=灵魂内部僭主——他被最低级的欲望绑架了一生——表面上享有一切——内里被自己从未被满足的欲望一直折磨
图6：个人正义与城邦正义的逻辑闭环——灵魂的和谐秩序=城邦的和谐秩序，基于《理想国》卷四核心论述复刻。核心推演：正义=秩序(Harmonia)→在灵魂中各部分安守其序→在城邦中各阶层安守其位。不正义=混乱(Stasis)→灵魂内部和城邦内部同时发生。柏拉图全书最深的论证结构：不是在比较两种「生活方式」，而是在把正义作为「灵魂的最健康状态」和「城邦的最健康状态」做统一的病理学剖析。一个正义的灵魂和一个正义的城邦同构于同一套「各安其位」的秩序逻辑。

📋 第七模块复盘

  • 灵魂正义=理性统领激情掌控欲望→内在和谐
  • 城邦正义=各阶层「各司其职」→城邦和谐
  • 正义者必比不义者幸福→因灵魂有秩序比灵魂有混乱更为幸福

八、理想秩序构建原则——从《理想国》到一切社会制度的元法则

柏拉图在《理想国》全书中为一切人类社会秩序提出了四条基础构建原则： 原则一·各安其位（τὰ ἑαυτοῦ πράττειν / "Minding One's Own Business"） 正义=每个人/每个阶层做它天生最适合做的事——绝不僭越越位 → 这违背现代「机会平等」，但它是柏拉图的「自然正义」 → 不是压制人→是顺应人的天然能力差异→使各方在最有利于自己和他人的分工中运行 原则二·精英统治（Aristocracy of Wisdom） 权力不应由「最有钱」「最有势」「最能煽动最多人」的人来行使 而应由「知道什么是善——并因此知道该为全社会做什么」的人来行使 → 现代自由的投票权=把决策权分配给每个个体(不论是否「知道什么对整体好」) → 柏拉图会问：让不知道真正的「善」究竟是啥的人投票←这是社会的福祉还是病症？ 原则三·教育即灵魂转向（Paideia → Periagoge） 教育的终极目的≠传播信息和技能——是把灵魂从感性世界的幻象中「扭过来」 对准纯粹理性的可知世界。教师的工作不是把视觉装入盲人的眼睛， 而是把已经从出生就拥有视觉、却一直盯着错误方向的灵魂转过来——对准真正的光源。 原则四·城邦的灵魂在于统治者的灵魂 一个城邦不会比统治它的人更好——「城邦的品质不是由制度、宪法或法律界定的， 而是由统治者的灵魂品质界定的。」制度≠运作自身的机器——制度是在人的灵魂之上运行的。

📋 第八模块复盘

  • 四元法则：各安其位·精英统治·灵魂教育·统治者的灵魂=城邦品质
  • 这四条原则是柏拉图对全部人类秩序的最抽象、最基本的自然正当性推证

九、教育哲学与灵魂塑造——柏拉图对人类可完善性的坚定信念

柏拉图教育哲学的三个核心主张——衍生出全部西方教育理念的底层原则： 主张一：知识=回忆（ἀνάμνησις / Anamnesis） 《美诺篇》+《理想国》中苏格拉底教一个从未学过几何的奴隶小孩推导出 正方形对角线与面积的关系——苏格拉底的结论：「他并没有被我『教』入任何新知识—— 他灵魂中原先就有的意见被正确的提问激活出来。学习=回忆。灵魂在进入身体之前 已经见过纯粹的理型——出生后的感官经验把这些原始知识覆盖了。教育的功能= 用正确的辩证法把这些被掩埋的真知从灵魂中重新唤醒。」 主张二：教育=欲望的重新定向（Reorientation of Desire） 「不是把视觉放入盲眼——是把一直有视觉但方向错了的眼睛转向光明。」 所有人在出生时都已拥有学习纯粹真理的能力。人们学不到的障碍不是「缺乏能力」—— 而是「欲望被错误的对象绑定了」：被绑在财富上的欲望→无法转向哲学； 被绑在荣誉上的欲望→无法转向真理。教育的功能=松开对错误对象的执着→ 让心灵的全部渴望自然地向新的、更高的对象(善)归位。 主张三：教育的终点=「善自身」的把握 数学·天文学·辩证法的全部训练只有一个最终顶点——让少数精英中的精英 「看见善的理型(τὸ ἀγαθόν)」——善是所有其他理型的存在和可知性的终极本原。 善在理型世界中的地位=太阳在可见世界中的地位——它是那使一切可见之物被看见，又使一切生长得以进行的光。

📋 第九模块复盘

  • 认知=回忆(先天理型知识被数学问题激活)
  • 教育=欲望的重新定向(从物质享受→转向纯粹善)
  • 终点的终点：善自身=「太阳」——切存在与可知性的终极光源

十、现代制度契约映射——柏拉图在今天

《理想国》十大原则在现代制度与社会中的映射： ① 精英治国 → 现代文官考试制度和专业技术官僚体系的底层哲学——标准≠财富或出身而是「所知道的」 ② 灵魂三分 → 现代心理动力学(弗洛伊德的id/ego/superego→本质上是柏拉图模型的改头换面) ③ 教育即灵魂转向 → 现代博雅教育(Liberal Arts)的核心理念——不是教一门谋生技能——而是教整个人的灵魂切换运行模式 ④ 正义=各安其位 → 现代宪法中的权力分立(行政·立法·司法)→每一支安守其宪法规定的职权边界 ⑤ 政体循环 → 政治学中「民主的危机」永续预言——过度自由最终召唤强人 ⑥ 洞穴寓言 → 媒介批判(大众媒体的「拟态环境」) + 互联网时代的「算法洞穴」 ⑦ 护卫者无私产 → 现代公职人员的「利益申报+回避制度」——权力者不可有额外私产以便公正无偏 ⑧ 哲人王 → 现代专家治国理想(Technocracy)—以专业的知识理性代替未经训练的多数民意做公共决策 ⑨ 金属神话 → 基本教育平等+先天潜力测试——现代精英选拔的底层假设 ⑩ 正义者必幸福 → 现代积极心理学：内在秩序感和自我和谐(Inner Congruence)是主观幸福感的最强预报因子

🏛️ 《理想国》全域总纲——正义·灵魂·秩序·哲人王的四重奏

全书的论证同心圆：正义之问(全书第一扳机)→灵魂三分(内心微观)→城邦构造(外在宏观)→护卫者教育(灵魂制造术)→洞穴寓言(认知宇宙的全景图)→政体循环(历史五步堕落)→正义者更幸福(终极价值判断)。柏拉图从未告诉你「现实社会应该怎样运行」——他问的是更根本的：「如果从人的灵魂的本质开始推演，一个终极正义的社会究竟可能长成什么样子？」

根本洞见——跨越两千四百年至今仍然炽热：制度的好坏取决于治理者灵魂的品质。一切制度的崩溃都不是因为法律条文不够精细——而是因为治理者的灵魂已经在权力中被腐蚀殆尽。正义不是一个外部规则——正义是人的灵魂内部每一部分在自己应处的位置上与全体和谐共振的最健康状态。当每个人的灵魂内部是正义的——城邦的正义尚未建立就已先有了它的微型的、但却是真实无伪的基础。
